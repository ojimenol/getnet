

# BalanceAmount

Amount.  The value uses the data format defined in ISO 20022 and has a maximum of 18 digits, of which 5 can be decimals, separated by a point. 

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**amount** | **Double** | Data in ISO 20022 format containing up to 18 digits from which 5 may be decimals.  Separation is a point  |  [optional] |
|**currency** | **String** | The currency code associated to the account. Follows ISO-4217. |  [optional] |
